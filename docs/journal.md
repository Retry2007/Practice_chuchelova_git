# Журнал проекта «Книга будущего»

## Этап 1 — Старт и организация

- Зарегистрирован Git-репозиторий проекта на GitHub
- Участники команды освоили базовые команды Git: клонирование, коммиты, ветки, пуши
- Определена и закреплена структура директорий репозитория
- Запущен Telegram-канал проекта с первыми публикациями
- Распределены роли внутри команды: текст/СММ, дизайн, подкаст, читательский клуб, художники

## Этап 2 — Контент и продвижение

- Сформулирована концепция Telegram-канала: факты о писателях, рецензии, опросы
- Начата работа над комиксом о чтении для дня открытых дверей
- Разработан макет платформы «ПолиКнига»
- Записаны первые выпуски подкаста — сценарии и монтаж
- Проведены первые встречи читательского клуба
- Создан мерч проекта (первая партия)

## Этап 3 — Сайт и визуальные материалы

- Создан статический сайт проекта на HTML и CSS
- Реализованы страницы: главная, «О проекте», «Участники», «Журнал», «Ресурсы»
- Добавлены графические элементы и иллюстрации книжной тематики
- Подготовлена документация по практике в формате Markdown

- Проведена ревизия и дополнение всей проектной документации

- Опубликовано более трёх выпусков подкаста
- Нарисовано шесть страниц комикса
