# О проекте

## Книга будущего

«Книга будущего» — студенческий проект Московского политехнического университета, направленный на популяризацию чтения среди молодёжи и продвижение книжной культуры в странах СНГ.

Проблема, на решение которой направлен проект: снижение интереса молодёжи к чтению в эпоху коротких видео и социальных сетей. Книги вытесняются из повседневной жизни, и команда проекта стремится изменить эту тенденцию через современные форматы коммуникации.

---

## Направления работы

- **Telegram-канал** — регулярные публикации: интересные факты о писателях, рецензии, опросы, викторины
- **Читательский клуб** — очные и онлайн-встречи, совместное обсуждение книг
- **Подкаст** — аудио-выпуски с разбором книг, интервью и дискуссиями
- **Мерч** — брендированная продукция с книжной тематикой
- **Листовки (лифлеты)** — иллюстрированные материалы, распространяемые на днях открытых дверей
- **Платформа «ПолиКнига»** — разрабатываемый сайт-агрегатор книжного контента
